# idlmatcher

**An Insights Hub VFC (Visualization Flow Component) flow project**

This repository contains a Node-RED flow implementation for the idlmatcher VFC component, part of the Siemens Insights Hub platform.

## Overview

idlmatcher is a visualization flow component designed for the Siemens Insights Hub. It leverages Node-RED to define data flow and processing logic for displaying insights and metrics.

## Project Structure

- `idlmatcher.json` - Node-RED flow definition

## Development

To work with this project locally:

1. Ensure you have Node-RED installed
2. Import the flow configuration from `idlmatcher.json`
3. Deploy and test the flow
